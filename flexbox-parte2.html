  • Main size: el tamaño con respecto al main axis por defecto es el width
  • Cross size: es el tamaño con respecto al cross axis, por defecto es el height
  • El tamaño de cross size se calcula bajo algunos escenarios:
    • Se ha definido un tamaño fijo por height
    • No se ha definido el tamaño ni align-items o align-content por lo que toma el valor de stretch
    • No se define un tamaño fijo pero se define align-items o align-content diferente de strech
  • El tamaño mdel main size se define por:
    1. El espacio disponible: es el espacio que no está ocupado por los items o sus márgenes
    2. El espacio ocupado: es el espacio que ocupan los items y sus márgenes
    3. Los márgenes no se colpasan
  • flex-basis: determina el tamaño base del item como si fuera width, su valor por defecto es auto
  • flex-grow: nos indica como va a crecer un elemento con respecto a su flex-basis, su valor por defecto es 0
  • flex-shrink: es lo contrario de flex-grow, lo que indica a un elemento que tanto a a encojerse cuando el edpacio disponible sea negativo, su valor por defecto es 1
  • flex: es un shorthan de flex-grow, flex-shrink y flex-basis. El valor predeterminado es 0 1 auto, llamado auto.
    • flex: auto = flex: 0 1 auto
    • flex: none = flex: 0 0 auto
    • flex: N = flex: N 0 0
