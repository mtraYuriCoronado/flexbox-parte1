Flexbox primera parte

  • Flexbox es un modelo de layout el cual define como se organiza y aparecen los elementos en la pantalla
  • Se utiliza la propiedad display: flex
  • Flexbox se aplica al contenedor padre para crear cajas flexibles como hijos
  • Cuando creamos el contexto flexible, se crean dos ejes: main axis y cross axis
  • Cambia la posici√≥n de los ejes: flex-direction: row(default ) | column | row-reverse | column-reverse
  • Alinear elementos en el main axis justify-content: flex-start(default) | center | space-between | space-around | space-evenly | flex-end
  • Alinea los elementos con respecto al cross axis align-items: stretch(default) | flex-start | flex-end | baseline | center
  • Alinea unicamente al elemento que se le aplica en el cross axis: align-self: mismos valores que align-items
